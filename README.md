# Capistrano Rails Console

Open a rails console the first app server.

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano_rails_console'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install capistrano_rails_console

## Usage

    $ cap rails:console
